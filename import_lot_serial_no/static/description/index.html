Import Serial Number and Lot Number in Picking Receipt

Import Lot Number
in Incoming
Shipment

Under Product > Inventory > Traceability set "Tracking" as "By Lots" to import product
with lot number on incoming shipment.

Now suppose you have made order of 22 quantity and its generated shipment for that purchase as shown in fig. Now if you want to Import it then you have to select the button which is highlighted in the fig.

Once you click on the button, you will see one wizard which is shown in the fig. Here you have to click on Import Lots/Serial Number Button.

Clicking the "Import Lot/Serial Number" button opens a wizard where you can choose to import either lot numbers or serial numbers based on your product's tracking configuration.

You can download sample file to import lot and serial number by enable "Download Sample Files" option and select "Sample Type" as CSV or XLS and click "DOWNLOAD SAMPLE FILE" button on wizard.

You can shows all set and all lot/serial numbers are imported with expiration date. Now you have to just confirm it.

On click of "Confirm" button you can see picking order's done quantity as per demanded quantity with lot number.

Import Serial
Number
in Incoming
Shipment

Under Product > Inventory > Traceability set "Tracking" as "By Unique Serial Number" to import product with serial number on incoming shipment.

Now suppose you have made order of 4 quantity and its generated shipment for that purchase as shown in fig. Now if you want to import product with serial number then you have to select the button which is highlighted in the fig.

Once you click on the button, You can see opened "Detailed Operations" wizard, here click on "Import Lot/Serial No" button.

On import wizard you have to select "Serial No" option and upload file containing serial number for product.

You can download sample file to import lot and serial number by enable "Download Sample Files" option and select "Sample Type" as CSV or XLS and click "DOWNLOAD SAMPLE FILE" button on wizard.

Once you click on "Import" button You can see serial number for product will imported with expiration date from excel file.

On click of "Confirm" button you can see picking order's done quantity as per demanded quantity with serial number.
